# Elson Gorica — Personal Portfolio (Theme v2)

Clean, responsive, accessible portfolio built with **HTML, CSS, and vanilla JS**.  
This theme uses a blue/cyan gradient style with glass panels.

## Pages
- index.html — Home
- about.html — About Me
- projects.html — Projects (carousel + cards)
- skills.html — Skills
- contact.html — Contact (form validation)

## Run locally
Open `index.html` or serve with `python -m http.server 3000` then visit `http://localhost:3000`.

## Deploy
- GitHub: push all files.
- Vercel: import repo → Other / static → root `/` → Deploy.
